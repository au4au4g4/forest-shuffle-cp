## CP 計算簡式

```text
總成本 = 費用 + 本身1張 + 回合2張 − 抽牌 − bonus／額外回合收益
CP = 得分 ÷ 總成本
```

---

## 核心套原則

```text
核心套 = 整組 CP ≥ 2，且是該流派第一次達標的最小組合。
如果小組合已經達標，就不再列同一路線繼續加牌的版本。
```
