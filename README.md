## 0. CP 計算規則

### 0.1 核心公式

```text
總成本 = 費用 + 本身1張 + 回合2張 − 抽牌 − bonus／額外回合收益
CP = 得分 ÷ 總成本
```

### 0.2 成本換算原則

| 項目 | 換算 |
|---|---:|
| 打出該牌本身 | +1 |
| 消耗 1 回合 | +2 |
| 費用 | +費用 |
| 抽 1 張牌 | −1 |
| 抽 2 張牌 | −2 |
| 額外 1 個完整回合 | 視為抽 2 張，−2 |
| 立即效果若視為抽牌 | 依抽牌數扣成本 |
| bonus 若抽牌 | 依抽牌數扣成本 |
| bonus 若給額外回合 | −2 |

### 0.3 表格表示方式

| 表示方式 | 意義 |
|---|---|
| `0 / 1.25` | 未達條件 CP / 達成條件 CP |
| `0.5 / 1 / 1.5 / 2 / 2.5` | 條件數 1/2/3/4/5 時的 CP |
| `5 / 3` | 未觸發效果成本 / 觸發效果後成本 |
| `功能牌` | 主要價值不只在分數，需搭配局勢判斷 |

# 10. 快速判讀

| CP 範圍 | 判讀 |
|---:|---|
| < 1 | 偏低，除非為了卡位、開位置或後續 combo |
| 1–1.5 | 可接受，通常是中期成形 |
| 1.5–2 | 不錯，值得投入 |
| 2–3 | 高 CP，通常是成熟主軸 |
| > 3 | 爆發型高 CP，常見於條件已成熟或 bonus 成功 |

## 10.1 高 CP 常見條件

| 條件 | 高 CP 牌 |
|---|---|
| 鳥符號 ≥ 4 | 蒼鷹 |
| 昆蟲符號 ≥ 4 | 紅腹灰雀 |
| 兩棲符號 ≥ 2 | 樹蕨 |
| 小飛蟲 ≥ 2 | 樹蛙 |
| 完整樹 ≥ 2 | 石貂 |
| 鹿 ≥ 2 | 狼 |
| 兔子 ≥ 4 | 赤狐＋兔子 |
| 已有 8 種樹 | 橡樹、野草莓 |
| 已有 10 棵樹 | 苔蘚 |
| 空地牌 ≥ 3 且棕熊 bonus | 棕熊 |
| 同一棵樹對面有蝙蝠 | 睡鼠 |
