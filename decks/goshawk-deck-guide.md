# 蒼鷹套完整攻略

> 本篇延伸 README 中的「蒼鷹套推薦」，用同一套 CP 計算邏輯整理。  
> 核心原則：蒼鷹本身不是前期單卡，而是鳥類符號累積到一定數量後的收割牌。

---

## 1. 蒼鷹核心定位

| 項目 | 內容 |
|---|---|
| 牌型定位 | 上方動物、鳥符號收割牌 |
| 費用 | 2 |
| 基礎總成本 | `2費用 + 1張本體 + 2張回合成本 = 5` |
| 計分 | 每個鳥類符號 3 分 |
| CP公式 | `CP = 鳥符號數 × 3 ÷ 5` |

蒼鷹的強度完全取決於鳥類符號數。  
若只有 1～2 個鳥符號，蒼鷹只是普通牌；到 3 個開始可打，4 個以上才是明顯高 CP。

### 蒼鷹套的真正邏輯

蒼鷹套不是單純「鳥很多就打」，而是：

```text
前期用鳥牌建立節奏與分數
→ 中期累積鳥符號
→ 中後期用蒼鷹把鳥符號轉成大量分數
→ 終局看是否能再補第二張蒼鷹或其他收割牌
```

## 2. 蒼鷹啟動 CP 表

| 鳥類符號數 | 得分 | 總成本 | CP | 判讀 |
|---:|---:|---:|---:|---|
| 0 | 0 | 5 | 0.00 | 不可打 |
| 1 | 3 | 5 | 0.60 | 偏低 |
| 2 | 6 | 5 | 1.20 | 可接受但不急 |
| 3 | 9 | 5 | 1.80 | 開始值得打 |
| 4 | 12 | 5 | 2.40 | 高 CP |
| 5 | 15 | 5 | 3.00 | 強力爆分 |
| 6 | 18 | 5 | 3.60 | 非常強 |
| 7 | 21 | 5 | 4.20 | 終局收割 |
| 8 | 24 | 5 | 4.80 | 爆發級 |
| 9 | 27 | 5 | 5.40 | 極高 |
| 10 | 30 | 5 | 6.00 | 天花板級 |

### 簡化判斷

| 狀態 | 建議 |
|---|---|
| 鳥符號 0～1 | 不建議打，除非只是為了卡位或啟動其他 combo |
| 鳥符號 2 | 可打但偏早，通常可以再等一張鳥 |
| 鳥符號 3 | 標準啟動點 |
| 鳥符號 4 | 強烈建議打 |
| 鳥符號 5+ | 終局核心收割牌 |

### 蒼鷹單卡門檻

| 目標 | 需要鳥符號數 | 原因 |
|---|---:|---|
| 不虧 | 2 | CP 1.2，至少不算差 |
| 值得打 | 3 | CP 1.8，已經高於一般可接受牌 |
| 主軸成立 | 4 | CP 2.4，足以作為中後期主收割 |
| 爆分 | 5+ | CP 3 以上，通常是終局關鍵分數 |

## 3. 蒼鷹套核心鳥牌

| 優先度 | 卡牌 | 單卡功能 | 對蒼鷹的價值 | 自身 CP |
|---:|---|---|---|---:|
| 1 | 灰林鴞 | 立即抽 1 張；固定 5 分；bonus 再抽 2 張 | 補牌、得分、提供鳥符號，最穩定 | 1.25 / 2.50 |
| 2 | 松鴉 | 完成本回合後再進行 1 個完整回合；每張 3 分 | 額外回合可補節奏，適合把蒼鷹延後成形 | 0.75 / 1.50 |
| 3 | 大斑啄木鳥 | 固定 1 分；若沒有其他玩家樹比你多，額外 10 分 | 條件成立時本身就是高 CP 鳥牌 | 0.25 / 2.75 |
| 4 | 紅腹灰雀 | 每個昆蟲符號 2 分 | 可與昆蟲流雙軸計分，順便增加鳥符號 | 0.5 / 1 / 1.5 / 2 / 2.5 |
| 5 | 蒼頭燕雀 | 放在山毛櫸上得 5 分 | 有山毛櫸才推薦，否則偏窄 | 0 / 1.25 |

> 注意：蒼鷹看的是「鳥類符號數」，不是單純看鳥類卡張數。實戰時仍以牌面圖示為準。

### 鳥牌分類

| 類型 | 代表牌 | 使用方式 |
|---|---|---|
| 節奏鳥 | 灰林鴞、松鴉 | 先打，幫你抽牌或多一回合，讓蒼鷹晚一點再爆分 |
| 爆分鳥 | 大斑啄木鳥、紅腹灰雀 | 條件成立時本身分數很高，蒼鷹只是額外收割 |
| 條件鳥 | 蒼頭燕雀 | 有山毛櫸才強，不建議硬湊 |
| 收割鳥 | 蒼鷹 | 等鳥符號成熟後再打 |

## 4. 推薦套牌版本

### 4.1 穩定抽牌版：灰林鴞＋松鴉＋蒼鷹

| 組件 | 功能 |
|---|---|
| 灰林鴞 | 抽牌、固定分、bonus 後成本很低 |
| 松鴉 | 額外回合，讓你有機會先補鳥符號再打蒼鷹 |
| 蒼鷹 | 中後期收割鳥符號 |

**特色**：最穩、不卡手，適合大多數局。  
灰林鴞與松鴉本身不只是在幫蒼鷹堆符號，也會改善手牌與回合節奏。

**適合情境**：

| 情境 | 判斷 |
|---|---|
| 手牌不穩 | 灰林鴞抽牌可以補手牌 |
| 想等蒼鷹成熟 | 松鴉額外回合能延後收割 |
| 鳥牌不多但品質高 | 靠高價值鳥牌撐住 CP |

---

### 4.2 爆分條件版：大斑啄木鳥＋樹木數優勢＋蒼鷹

| 組件 | 功能 |
|---|---|
| 大斑啄木鳥 | 若樹木數不落後，單卡可到 11 分 |
| 多樹策略 | 支援大斑啄木鳥條件，也增加上方位置 |
| 蒼鷹 | 把已經打出的鳥符號再轉成分數 |

**特色**：分數高，但需要維持樹木數不輸人。  
若你本來就走多樹、橡樹、槭樹／楓樹、苔蘚等路線，大斑啄木鳥會很自然地變強。

**適合搭配的樹與下方牌**：

| 搭配 | 理由 |
|---|---|
| 樺木 | 0 費、抽 1，容易鋪樹不虧 |
| 槭樹／楓樹 | 森林越大越強，與多樹方向一致 |
| 橡樹 | 多樹種時可走 8 種樹路線 |
| 苔蘚 | 已有 10 棵樹時高 CP |
| 野草莓 | 若同時達 8 種樹，能補高分 |

---

### 4.3 山毛櫸支線：山毛櫸＋蒼頭燕雀＋蒼鷹

| 組件 | 功能 |
|---|---|
| 山毛櫸 | 支援蒼頭燕雀條件，也可走 4 棵山毛櫸計分 |
| 蒼頭燕雀 | 放在山毛櫸上得 5 分 |
| 蒼鷹 | 把蒼頭燕雀的鳥符號納入收割 |

**特色**：條件較窄，但若你已經有山毛櫸，蒼頭燕雀會從雞肋牌變成可用牌。  
不建議為了蒼頭燕雀硬走山毛櫸；應該是「已有山毛櫸時順手拿」。

**判斷**：

| 山毛櫸狀態 | 蒼頭燕雀建議 |
|---|---|
| 沒有山毛櫸 | 不拿或不打 |
| 只有 1 棵山毛櫸 | 可考慮，但不是優先 |
| 已有 2～3 棵山毛櫸 | 可以納入支線 |
| 已接近 4 棵山毛櫸 | 蒼頭燕雀與山毛櫸都可變成穩定得分 |

---

### 4.4 昆蟲混合版：紅腹灰雀＋昆蟲符號＋蒼鷹

| 組件 | 功能 |
|---|---|
| 紅腹灰雀 | 每個昆蟲符號 2 分 |
| 昆蟲牌 | 支援紅腹灰雀，也可能支援其他昆蟲相關 combo |
| 蒼鷹 | 把紅腹灰雀本身的鳥符號再轉成分數 |

**特色**：雙軸計分，不完全依賴鳥牌。  
當你已經有 3 個以上昆蟲符號時，紅腹灰雀本身就值得打；蒼鷹只是額外把牠納入鳥流收割。

**昆蟲混合版的重點**：

| 昆蟲符號數 | 紅腹灰雀得分 | 紅腹灰雀 CP | 判斷 |
|---:|---:|---:|---|
| 1 | 2 | 0.5 | 不夠強 |
| 2 | 4 | 1.0 | 勉強可用 |
| 3 | 6 | 1.5 | 可打 |
| 4 | 8 | 2.0 | 高 CP |
| 5 | 10 | 2.5 | 強 |

紅腹灰雀本身若已經達 3～4 個昆蟲符號，就算沒有蒼鷹也可用；若同時被蒼鷹計入鳥符號，等於一張牌被兩套系統重複利用。

---

### 4.5 終局收割版：先忍住蒼鷹，最後一次爆分

| 組件 | 功能 |
|---|---|
| 前期高價值鳥牌 | 灰林鴞、松鴉、大斑啄木鳥 |
| 中期補牌或多樹 | 增加鳥牌上場機會與放置位置 |
| 終局蒼鷹 | 用 4～7 個鳥符號一次收割 |

**特色**：最符合蒼鷹的牌性。  
蒼鷹越晚打，通常 CP 越高，但太晚可能沒回合或沒位置，所以要抓中後期節奏。

| 剩餘回合感 | 建議 |
|---|---|
| 還很早 | 蒼鷹先留，繼續補鳥牌 |
| 中期已 3 鳥 | 可以打，尤其手牌缺分時 |
| 中後期 4 鳥 | 通常直接打 |
| 終局 5+ 鳥 | 高優先打，除非有更高 CP combo |

---

### 4.6 雙軸混合版：蒼鷹＋另一個主軸

蒼鷹套最穩的玩法通常不是純鳥，而是「鳥流＋另一個流派」。因為鳥牌數量有限，硬湊純鳥常常會卡手。

| 第二主軸 | 推薦度 | 理由 |
|---|---:|---|
| 多樹／8 種樹 | ★★★★★ | 支援大斑啄木鳥、橡樹、野草莓、苔蘚 |
| 昆蟲 | ★★★★☆ | 紅腹灰雀可雙重利用 |
| 山毛櫸 | ★★★☆☆ | 蒼頭燕雀可用，但條件較窄 |
| 蝴蝶 | ★★★☆☆ | 若同時有紅腹灰雀，可形成昆蟲支線 |
| 兔子／赤狐 | ★★☆☆☆ | 不直接支援鳥，但可作為另一個分數來源 |
| 鹿／狼 | ★★☆☆☆ | 強但吃手牌與費用，容易與蒼鷹搶資源 |

## 5. 整組 CP 範例

以下用「整組總分 ÷ 整組總成本」估算，會把前置鳥牌成本也算進去，所以 CP 會比只算蒼鷹啟動牌低。

| 組合 | 鳥符號數 | 假設 | 整組成本 | 整組得分 | 整組 CP |
|---|---:|---|---:|---:|---:|
| 蒼鷹＋灰林鴞 | 1 | 灰林鴞未 bonus | 9 | 8 | 0.89 |
| 蒼鷹＋灰林鴞 bonus | 1 | 灰林鴞 bonus 抽 2 張 | 7 | 8 | 1.14 |
| 蒼鷹＋灰林鴞 bonus＋松鴉效果 | 2 | 灰林鴞 bonus、松鴉額外回合 | 9 | 14 | 1.56 |
| 蒼鷹＋灰林鴞 bonus＋松鴉＋大斑啄木鳥失敗 | 3 | 大斑啄木鳥只有 1 分 | 13 | 18 | 1.38 |
| 蒼鷹＋灰林鴞 bonus＋松鴉＋大斑啄木鳥成功 | 3 | 大斑啄木鳥 11 分 | 13 | 28 | 2.15 |
| 上列＋蒼頭燕雀成功 | 4 | 蒼頭燕雀放山毛櫸 | 17 | 36 | 2.12 |
| 上列＋紅腹灰雀，且昆蟲符號 3 個 | 5 | 紅腹灰雀得 6 分 | 21 | 45 | 2.14 |
| 灰林鴞 bonus＋松鴉效果＋紅腹灰雀 4 昆蟲＋蒼鷹 | 3 | 紅腹灰雀 8 分 | 13 | 25 | 1.92 |
| 灰林鴞 bonus＋松鴉效果＋大斑啄木鳥成功＋紅腹灰雀 4 昆蟲＋蒼鷹 | 4 | 雙高分鳥 | 17 | 40 | 2.35 |
| 灰林鴞 bonus＋松鴉效果＋大斑啄木鳥成功＋紅腹灰雀 4 昆蟲＋蒼頭燕雀成功＋蒼鷹 | 5 | 完整鳥流 | 21 | 48 | 2.29 |

### 解讀

- **只靠 1～2 張鳥硬打蒼鷹，整組 CP 不一定好。**
- 蒼鷹套真正強的地方，是前置鳥牌本身也有分數或抽牌價值。
- 大斑啄木鳥成功、灰林鴞 bonus、松鴉額外回合，是蒼鷹套最容易把整組 CP 拉高的關鍵。
- 紅腹灰雀若同時吃到 3～4 個昆蟲符號，會讓蒼鷹套變成雙軸高分。

## 6. 打牌順序建議

| 階段 | 建議 |
|---|---|
| 前期 | 先打灰林鴞、松鴉這類不卡手鳥牌；蒼鷹通常先留著 |
| 中期 | 鳥符號達 3 個左右，開始評估是否打蒼鷹 |
| 中後期 | 若手上還有鳥牌、松鴉或灰林鴞 bonus，可先補符號再打蒼鷹 |
| 終局 | 鳥符號 4～6 個時，蒼鷹通常是高 CP 收割牌 |

### 最佳順序

```text
先建立有價值的鳥牌
→ 用灰林鴞／松鴉維持手牌與回合節奏
→ 鳥符號達 3～4 個
→ 打蒼鷹收割
→ 若還有額外回合或抽牌，再補第二波分數牌
```

### 不同手牌下的順序

| 手牌狀態 | 建議順序 |
|---|---|
| 有灰林鴞＋蒼鷹 | 先灰林鴞，蒼鷹等第 2～3 鳥 |
| 有松鴉＋蒼鷹 | 先松鴉，利用額外回合找或打更多鳥 |
| 有大斑啄木鳥＋蒼鷹 | 先確認樹木數是否能不輸人，再打啄木鳥 |
| 有紅腹灰雀＋蒼鷹 | 先看昆蟲符號數；昆蟲 ≥3 再打紅腹灰雀 |
| 有蒼頭燕雀＋蒼鷹 | 先看山毛櫸；沒有山毛櫸就不要硬走 |
| 只有蒼鷹 | 通常先不要打，除非終局沒其他分數牌 |

## 7. 起手與保留建議

| 牌 | 起手建議 | 理由 |
|---|---|---|
| 灰林鴞 | 優先保留 | 抽牌＋固定 5 分，最穩 |
| 松鴉 | 優先保留 | 額外回合價值高，可幫蒼鷹套成形 |
| 蒼鷹 | 有其他鳥牌才保留 | 單獨留容易卡手 |
| 大斑啄木鳥 | 視樹木策略保留 | 若能維持樹木數不輸人，價值很高 |
| 蒼頭燕雀 | 有山毛櫸才保留 | 條件太窄，不宜硬拿 |
| 紅腹灰雀 | 有昆蟲符號才保留 | 走昆蟲混合流時才高 CP |

### 起手保留公式

| 起手組合 | 建議 |
|---|---|
| 蒼鷹單張 | 不一定留，除非其他牌很弱 |
| 蒼鷹＋灰林鴞 | 可以留，穩定起手 |
| 蒼鷹＋松鴉 | 可以留，節奏好 |
| 蒼鷹＋灰林鴞＋松鴉 | 很好的蒼鷹套起手 |
| 蒼鷹＋大斑啄木鳥＋低費樹 | 可留，走多樹啄木鳥線 |
| 蒼鷹＋紅腹灰雀但無昆蟲 | 普通，不要硬走 |
| 蒼鷹＋蒼頭燕雀但無山毛櫸 | 偏弱，不建議 |

## 8. 實戰決策表

### 8.1 看到蒼鷹要不要拿？

| 當下狀態 | 建議 |
|---|---|
| 已有 0 個鳥符號 | 通常不拿，除非手牌缺主軸 |
| 已有 1 個鳥符號 | 可觀望，不急著拿 |
| 已有 2 個鳥符號 | 可以拿，準備中期啟動 |
| 已有 3 個鳥符號 | 建議拿，已達啟動點 |
| 已有 4+ 個鳥符號 | 高優先拿，通常是爆分牌 |
| 手牌有灰林鴞／松鴉 | 蒼鷹價值上升 |
| 場上鳥牌位置快滿 | 蒼鷹要提早規劃，不要等到沒位置 |

### 8.2 蒼鷹要不要立刻打？

| 狀況 | 建議 |
|---|---|
| 鳥符號 1 | 不打 |
| 鳥符號 2 | 除非終局，不然再等 |
| 鳥符號 3 | 可以打，尤其缺分時 |
| 鳥符號 4 | 通常打 |
| 鳥符號 5+ | 高優先打 |
| 手上還有灰林鴞或松鴉可先打 | 先打牠們，再打蒼鷹 |
| 剩餘回合不多 | 不要貪，達 3～4 鳥就打 |

### 8.3 蒼鷹與其他高 CP 牌的比較

| 牌 | 高 CP 條件 | 跟蒼鷹比較 |
|---|---|---|
| 樹蕨 | 兩棲符號 ≥1 | 樹蕨更早啟動，蒼鷹更偏中後期 |
| 樹蛙 | 小飛蟲符號 ≥1 | 樹蛙爆發更快，但需要特定符號 |
| 狼 | 鹿 ≥2 | 狼上限更誇張，但費用與條件更重 |
| 赤狐 | 兔子 ≥3 | 赤狐吃兔子數，蒼鷹吃鳥符號數 |
| 棕熊 | 空地牌多且 bonus | 棕熊偏洞穴與爆發，蒼鷹偏穩定收割 |
| 蒼鷹 | 鳥符號 ≥4 | 條件清楚，成熟後 CP 穩定高 |

## 9. 兩人局／多人局差異

| 局型 | 蒼鷹套變化 |
|---|---|
| 兩人局 | 牌池競爭較低，較容易慢慢等鳥牌；蒼鷹可晚一點打 |
| 三人局 | 鳥牌被拿走機率上升，看到灰林鴞、松鴉要更積極 |
| 四人局 | 等太久可能湊不到鳥，蒼鷹達 3～4 鳥就可出手 |

### 兩人局建議

| 策略 | 說明 |
|---|---|
| 可以貪一點 | 因為被搶牌壓力較小，蒼鷹可等到 4～5 鳥 |
| 多樹線更穩 | 大斑啄木鳥條件比較容易控制 |
| 注意終局節奏 | 不要為了等第 5 鳥，結果蒼鷹沒打出去 |

### 多人局建議

| 策略 | 說明 |
|---|---|
| 看到高價值鳥先拿 | 灰林鴞、松鴉、大斑啄木鳥容易被搶 |
| 蒼鷹不要等太久 | 3～4 鳥已經可以啟動 |
| 避免過度依賴山毛櫸支線 | 條件太窄，多人局更容易被打亂 |

## 10. 蒼鷹套與樹木選擇

蒼鷹是上方動物，所以需要樹的位置。樹的選擇會影響鳥牌能否順利上場。

| 樹木 | 搭配度 | 理由 |
|---|---:|---|
| 樺木 | ★★★★★ | 低成本、抽牌，適合前期開位置 |
| 槭樹／楓樹 | ★★★★☆ | 森林越大越強，與多樹鋪場一致 |
| 橡樹 | ★★★★☆ | 可與 8 種樹、野草莓支線合流 |
| 山毛櫸 | ★★★☆☆ | 支援蒼頭燕雀，但要達 4 棵才有主軸價值 |
| 花旗松 | ★★★☆☆ | bonus 額外回合可幫助補鳥與打蒼鷹 |
| 銀冷杉 | ★★☆☆☆ | 看位置與附著牌，不是蒼鷹套核心 |
| 椴樹 | ★★☆☆☆ | 分數普通，但便宜可開位置 |
| 七葉樹 | ★★☆☆☆ | 自己有套組壓力，與蒼鷹不是特別合 |

### 推薦樹木方向

```text
穩定版：樺木 + 槭樹／楓樹
爆分版：樺木 + 橡樹 + 多樹種
山毛櫸支線：山毛櫸 + 蒼頭燕雀
節奏版：花旗松 + 松鴉 + 蒼鷹
```

## 11. 常見錯誤

| 錯誤 | 問題 | 修正 |
|---|---|---|
| 太早打蒼鷹 | 1～2 個鳥符號時 CP 不高 | 至少等 3 個鳥符號，4 個更好 |
| 為了蒼鷹硬打低價值鳥牌 | 前置成本過高，整組 CP 下降 | 優先選本身就有分數、抽牌或額外回合的鳥 |
| 忽略灰林鴞 bonus | 灰林鴞 bonus 後成本大幅下降 | 能觸發時優先規劃 |
| 硬走蒼頭燕雀 | 沒山毛櫸時幾乎無分 | 只在已有山毛櫸時加入 |
| 只看蒼鷹單卡 CP | 忽略前置鳥牌成本 | 同時看整組 CP |
| 位置開太少 | 鳥牌想打卻沒上方位置 | 前期先用低成本樹開位置 |
| 過度貪高鳥數 | 等到 5～6 鳥但沒時間打 | 中後期 4 鳥通常就可收割 |
| 純鳥硬湊 | 鳥牌品質不夠時會卡手 | 改走鳥＋多樹或鳥＋昆蟲雙軸 |

## 14. 一句話結論

蒼鷹套不是「看到蒼鷹就硬打」，而是先用灰林鴞、松鴉、大斑啄木鳥等高價值鳥牌建立分數與節奏，搭配多樹、昆蟲或山毛櫸支線，等鳥符號達 3～4 個後，再用蒼鷹把整套牌一次收割成高 CP。
