# 樹蕨套攻略

## 核心概念

樹蕨套是兩棲符號收割套。

樹蕨本身會立即抽 1 張，因此成本低；只要場上有 1 個兩棲符號，樹蕨就能達到 CP 2。若有 2 個以上兩棲符號，樹蕨會變成高 CP 收割牌。

整理時採用最小達標組合原則：

```text
只列第一次 CP > 2 的組合。
如果小組合已經達標，就不再列同一路線繼續加牌的版本。
```

---

## CP 快表

| 組合 | 分數 | CP | 評價 |
|---|---:|---:|---|
| **1 火蠑螈 + 樹蕨** | 11 | **2.20** | 最小達標 |
| **2 普通蟾蜍 + 樹蕨** | 22 | **2.44** | 推薦 |
| **3 樹蛙 + 2 樹蕨** | 36 | **2.40** | 2 樹蕨路線 |

| 組合 | 不列原因 |

| 2 火蠑螈 + 樹蕨 | 1 火蠑螈 + 樹蕨已達標 |
| 3 火蠑螈 + 樹蕨 | 1 火蠑螈 + 樹蕨已達標 |
| 2 普通蟾蜍 + 2 樹蕨 | 2 普通蟾蜍 + 樹蕨已達標 |
| 1 火蠑螈 + 2 樹蕨 | 沒有比 1 火蠑螈 + 樹蕨更好 |

---

## 2 樹蕨判斷

2 樹蕨不是看到就打，而是要有足夠兩棲符號才值得。

| 兩棲基底 | 2 樹蕨評價 |
|---|---|
| 1 個兩棲 | 不值得 |
| 2 個兩棲 | 勉強可接受 |
| 3 個兩棲 | 開始值得 |
| 4 個以上兩棲 | 很強，但通常已不是最小組合 |

## 抓牌優先度

★★★★★
- 樹蕨
- 火蠑螈
- 第 2 張普通蟾蜍

★★★★☆
- 第 1 張普通蟾蜍
- 第 2 張樹蕨（已有 3 個以上兩棲時）

★★★☆☆
- 樹蛙

---

## 一句話攻略

> **樹蕨套只需要找最小達標組合。**  
> 1 火蠑螈 + 樹蕨已經夠強；2 樹蕨主要留給 3 樹蛙這種低分兩棲基底。
