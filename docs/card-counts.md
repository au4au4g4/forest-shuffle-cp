# 基本版卡牌數量表

> 本表只統計《森森不息 / Forest Shuffle》**基本版**。  
> 「張數」是該樹木或生物在牌組中實際出現的次數；分割卡以可打出的半邊卡面分別計算。

---

## 1. 牌組結構

| 類型 | 實體卡數 | 可打出的卡面數 |
|---|---:|---:|
| 完整樹木卡 | 66 | 66 |
| 上／下分割卡 | 48 | 96 |
| 左／右分割卡 | 44 | 88 |
| **合計** | **158** | **250** |

- 共 **57 種**不同樹木／生物：8 種樹木、12 種上方生物、17 種下方生物、20 種左右生物。
- 冬季卡、洞穴卡與玩家幫助卡不列入本表。
- 分割卡的兩個生物共享同一張實體卡，因此「卡面數」不等於牌組中的實體卡數。

## 6. 實戰記憶重點

- 最多：七葉樹、歐洲野兔，各 **11 張**。
- 所有上方生物均為 **4 張**。
- 四種蝙蝠各 **3 張**，牌組中共有 **12 個蝙蝠卡面**。
- 下方牌中最稀有的是雞油菌、毒蠅傘、鼴鼠、大傘蘑菇、美味牛肝菌、澤龜與鍬形蟲，各 **2 張**。
- 判斷「牌是否可能再出現」時，要記得兩個生物共用一張分割卡；其中一邊進入森林或棄牌區，另一邊也會同時離開牌庫。
