## 6. 常用高 CP 條件

| 條件 | 高 CP 牌 |
|---|---|
| 鳥符號 ≥ 4 | 蒼鷹 |
| 昆蟲符號 ≥ 4 | 紅腹灰雀 |
| 兩棲符號 ≥ 2 | 樹蕨 |
| 小飛蟲 ≥ 2 | 樹蛙 |
| 完整樹 ≥ 2 | 石貂 |
| 鹿 ≥ 2 | 狼 |
| 兔子 ≥ 4 | 赤狐＋兔子 |
| 已有 8 種樹 | 橡樹、野草莓 |
| 已有 10 棵樹 | 苔蘚 |
| 同一棵樹對面有蝙蝠 | 睡鼠 |
| 沒有其他玩家森林有更多樹 | 大斑啄木鳥 |
