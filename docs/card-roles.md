# 卡牌定位整理

> 先依目前 repo 已整理到 `docs/card-cp-reference.md` 的卡牌分類。  
> 完整全卡定位待全卡表補齊後再擴充。

---

## 1. Combo 核心

這類牌通常需要前置牌配合，形成一條明確路線。

| 卡牌 | 主要路線 | 說明 |
|---|---|---|
| 蒼鷹 | 鳥套 | 鳥符號達 4 以上時成為高 CP 收割 |
| 紅腹灰雀 | 昆蟲套 | 昆蟲符號達 4 以上時達標 |
| 樹蕨 | 兩棲套 | 兩棲符號達 2 以上時爆分 |
| 樹蛙 | 小飛蟲套 | 小飛蟲符號達 2 以上時爆分 |
| 火蠑螈 | 兩棲／獸族連打 | 自己有分，bonus 可連打獸族 |
| 飛螢 | 飛螢套 | 不同飛螢成套計分 |
| 歐洲野兔 | 野兔套 | 兔子數量平方計分 |
| 赤狐 | 野兔套 | 兔子數量越多，抽牌與得分越高 |
| 狼 | 鹿套 | 鹿數量達 2 以上時高 CP |
| 黇鹿 | 偶蹄動物套 | 偶蹄動物數量越多越強 |
| 狍鹿 | 同樹種符號套 | 同樹種符號越多越強 |
| 猞猁／山貓 | 狍鹿套 | 有狍鹿時直接 10 分 |
| 蝙蝠系列 | 蝙蝠套 | 至少 3 種不同蝙蝠才啟動 |
| 睡鼠 | 蝙蝠睡鼠線 | 同樹對面有蝙蝠時爆分 |

---

## 2. 收割核心

這類牌本身是收割牌，條件成熟後單張就能達標。

| 卡牌 | 核心條件 | 說明 |
|---|---|---|
| 大斑啄木鳥 | 樹數不落後 | 成功時 10 分，成本低 |
| 野草莓 | 8 樹種 | 成功時 10 分，成本低 |
| 橡樹 | 8 樹種 | 成功時 10 分 |
| 苔蘚 | 10 棵樹 | 成功時 10 分，成本低 |
| 黑莓 | 植物 ≥ 3 | 植物符號收割 |
| 木蟻 | 樹下方牌 ≥ 4 | 下方牌越多越強 |
| 石貂 | 完整樹 ≥ 2 | 完整樹收割，高 CP |
| 野豬 | 有小菇 | 成功時 10 分 |
| 浣熊 | 洞穴牌 | 洞穴牌越多越強 |
| 棕熊 | 洞穴牌 | bonus 後成本低，可作洞穴收割 |
| 槭樹／楓樹 | 總卡牌數 | 森林越大越強 |
| 銀冷杉 | 附著牌 | 四周附著越滿越高分 |
| 山毛櫸 | 山毛櫸數量 | 至少 4 棵後才啟動 |
| 七葉樹 | 七葉樹數量 | 數量平方成長 |
| 椴樹 | 椴樹數量比較 | 多數或並列多數才有效率 |

## 3. 引擎核心

這類牌主要不是自己得分，而是降低成本、補牌或延長節奏。

| 卡牌 | 用途 | 說明 |
|---|---|---|
| 鼴鼠 | 連打引擎 | 一次打出多張牌，省回合成本 |
| 松鴉 | 額外回合 | 額外完整回合視為降低成本 |
| 灰林鴞 | 補牌引擎 | 立即抽牌，bonus 再抽 2 張 |
| 花旗松 | 額外回合 | bonus 可提供額外回合 |
| 歐洲獾 | 免費打獸族 | bonus 可免費放獸族／爪印動物 |
| 小飛蟲 | 蝙蝠引擎 | 可免費放任意數量蝙蝠 |
| 紫木蜂 | 樹種判定 | 增加指定樹種數量判定 |
| 樺木 | 補牌樹 | 立即抽 1，低成本開局 |
| 浣熊 | 換牌／洞穴 | 把手牌放入洞穴並補牌 |

---

## 4. 低效或偏輔助

這類牌通常不能單獨成為核心，但可作為補位、符號來源或路線零件。

| 卡牌 | 定位 | 說明 |
|---|---|---|
| 蝴蝶系列 | 符號／副套 | 5 蝴蝶本身 CP 不高，需搭鼴鼠、紅腹灰雀或刺蝟 |
| 蒼頭燕雀 | 山毛櫸支線 | 有山毛櫸才考慮 |
| 赤松鼠 | 橡樹支線 | 有橡樹才考慮 |
| 普通蟾蜍 | 兩棲基底 | 成對後可用，主要支援樹蕨 |
| 馬蹄蝠等蝙蝠 | 蝙蝠基底 | 需 3 種不同蝙蝠才有分 |
| 歐洲獾未 bonus | 補位 | 未 bonus 時效率低 |
