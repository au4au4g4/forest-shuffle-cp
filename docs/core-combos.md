# 核心套整理

## 定義

核心套 = **整組 CP ≥ 2**，且是該流派值得記的達標組合。

```text
總成本 = 費用 + 本身1張 + 回合2張 − 抽牌 − bonus／額外回合收益
CP = 得分 ÷ 總成本
```

補充規則：

```text
鼴鼠一次最多額外打出 5 張牌。
超過 5 張的組合，不列入鼴鼠核心套。

若條件本身是「必須為此組合額外打出的特定牌」，要把該牌成本一起算進整組成本。
若條件是既有場面、符號數、樹數、洞穴牌數，收割牌可只算自身成本。
```
