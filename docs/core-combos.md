# 核心套整理

## 定義

核心套 = **整組 CP ≥ 2**，且是該流派值得記的達標組合。

```text
總成本 = 費用 + 本身1張 + 回合2張 − 抽牌 − bonus／額外回合收益
CP = 得分 ÷ 總成本
```

補充規則：

```text
鼴鼠一次最多額外打出 5 張牌。
超過 5 張的組合，不列入鼴鼠核心套。

若條件本身是「必須為此組合額外打出的特定牌」，要把該牌成本一起算進整組成本。
若條件是既有場面、符號數、樹數、洞穴牌數，收割牌可只算自身成本。
* 代表 bonus 後成本。
```

## 樹木暫不列入核心套

| 樹木 | 原因 |
|---|---|
| 樺木 | 固定 1 分，成本 2，CP 0.50 |
| 山毛櫸 | 4 棵後每棵 5 分，但完整成本 4/張，整組 CP 約 1.25 |
| 銀冷杉無 bonus | 4 格滿時 8 分，成本 5，CP 1.60 |
| 花旗松 | 成本 5/3*，固定 5 分，CP 1/1.67* |
| 七葉樹 | n 張 CP = n/4，通常最高約 1.75；需 8 張才達 2 |
| 椴樹 | 即使最多或並列最多，每張 CP 仍約 0.75 |
