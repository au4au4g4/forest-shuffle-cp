# 核心套整理

## 定義

核心套 = **整組 CP ≥ 2**，且是該流派值得記的達標組合。

```text
總成本 = 費用 + 本身1張 + 回合2張 − 抽牌 − bonus／額外回合收益
CP = 得分 ÷ 總成本
```

補充規則：

```text
鼴鼠一次最多額外打出 5 張牌。
超過 5 張的牌，不能再吃同一張鼴鼠的省回合效果，需另外計算回合成本。

Combo 核心套與收割型核心套合併整理；
同一張牌若已在核心套表內出現，就不再另外列泛用收割門檻。
```
