## 核心套定義

> 核心套 = 整組 CP ≥ 2，且是該流派第一次達標的最小組合。

## 3. 引擎型核心

| 卡牌 | 用途 |
|---|---|
| 鼴鼠 | 一次打出多張牌，降低整組成本 |
| 松鴉 | 額外完整回合，提高節奏 |
| 火蠑螈 | bonus 打出下一張獸族，降低成本 |
| 歐洲獾 | bonus 打出高成本獸族，形成連打 |
